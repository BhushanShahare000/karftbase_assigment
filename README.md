## 🛠️ Tech Stack
- **Next.js 14**
- **React.js**
- **Tailwind CSS**
- **TypeScript (optional if used)**
- **Vercel (Deployment)**

---

## 📋 Features
- Fully responsive (mobile, tablet, desktop)
- Pixel-accurate Figma implementation
- Hover effects and subtle transitions
- Scrollable testimonial carousel
- Gradient + blur background animations
- Clean and modular code using React components

## ⚙️ Setup Instructions

1. **Clone the Repository**
   ```bash
   git clone https://github.com/yourusername/kraftbase-assignment.git
   cd kraftbase-assignment
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Run the Development Server**
   ```bash
   npm run dev
   ```

4. **Open in Browser**
   Visit [http://localhost:3000](http://localhost:3000)
